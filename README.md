

## Начало

Запустить сервер:

```bash
npm i
npm run dev
# or
yarn add
yarn dev
# or
pnpm i 
pnpm dev
# or
не знаю
bun dev
```

Открой [http://localhost:3000](http://localhost:3000) что бы увидеть сайт.
